## 四、拍摄统一规范

- **分辨率**：窗口宽度 ≥ 1400px（Retina 可 2x）；导出 PNG。
- **主题**：README 主视觉建议统一 **dark** 或 **midnight**（与 GitHub 深色预览协调）；画廊可混 light。
- **隐私**：对话/群聊/蒸馏材料打码；Nexus 昵称用测试实例名。
- **版本**：用当前安装包（如 `dist-electron/mac-arm64/灵犀.app`），避免浏览器 dev 与正式 UI 不一致。
- **群聊**：展示 2026-05 后行为（少 @ 用户、无 thinking、无 `[捂脸]` 文字表情）。
